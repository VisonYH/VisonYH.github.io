##### 初始化：

```go
// 1基本方式
a1 = [3]bool{true, true, true}

// 2 根据初始值自动推断数组长度
a10 = [...]int{0,1,2,3,4,5}

//根据索引初始化
a3 = [5]int{0:1, 4: 2}
```



##### 遍历

```go
cities := [...]string{"北京", "上海", "深圳"}

// 1、根据索引遍历
for i:= 0;i<len(cities); i++{
  fmt.Println(cities[i])
}

// 2、 for range遍历
for _, v := range cities {
  fmt.Println(v)
}


```
